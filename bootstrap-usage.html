Layout:
container-fluid: For a full-width container, spanning the entire width of the viewport.
row: For creating horizontal groups of columns.
col-md-4, col-lg-3, col-md-12, col-md-6, col-sm-6, etc.: For responsive column layouts based on device or viewport size.

Content:
text-center: For center-aligned text.
text-left: For left-aligned text.
text-muted: For muted text color.
text-white: For white text color.

Components:
navbar: The data attributes like data-spy, data-target, and data-offset for a Bootstrap navbar or scrollspy implementation.
badge: For displaying count and labeling.
badge-pill: For pill-shaped badges.
badge-warning: For yellow/orange badges.
list-group: For a list group.
list-group-item: For list items in a list group.
list-group-item-action: For actionable list group items.
card: For card components.
card-header: For the header of a card.
card-body: For the body of a card.
card-footer: For the footer of a card.
card-title: For the title within a card.
card-text: For text content within a card.
card-img-top: For top-aligned images in a card.
img-fluid: For responsive images that scale with the parent element.
rounded-circle: For round images.
btn: For buttons.
btn-warning: For yellow/orange buttons.
modal: For modals.
modal-dialog: For dialog sizing and positioning within a modal.
modal-content: For the content of a modal.
modal-header: For the header of a modal.
modal-body: For the body of a modal.
modal-footer: For the footer of a modal.
modal-title: For the title within a modal.
progress: For progress bars.
progress-bar: For the actual progress bar element.
progress-bar-animated: For animated stripes on progress bars.
carousel: For image sliders/carousels.
carousel-inner: For the inner part of a carousel that holds slides.
carousel-item: For individual items in a carousel.
carousel-control-prev: For the previous control of a carousel.
carousel-control-next: For the next control of a carousel.

Utilities:
d-flex: For flexbox layout.
justify-content-between: For space between flex items.
align-items-center: For vertically centering flex items.
float-right: For floating elements to the right.
float-left: For floating elements to the left.
clearfix: To clear the float.
mb-0, mb-3, mb-5, mt-5, etc.: For margin bottom and margin top spacing.
ml-5, mr-5: For margin left and right spacing.

Javascript Components (via data attributes):
data-toggle="list": For toggling list groups.
data-toggle="tab": For toggling tabs.
data-toggle="modal": For toggling modals.
data-slide: For carousel slide actions.
data-ride="carousel": For initiating a carousel on load.

Icons:
bi-*: Bootstrap icons classes are used throughout the document.
