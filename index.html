Blazor WebAssembly Webpage

In my 'MatchScore' project, I developed a sophisticated Blazor WebAssembly application that delivers live football match scores and detailed event data to users. This front-end interface is supported by a robust backend service, FootballApiService, which communicates with an external football API to retrieve and process information such as fixtures, team statistics, league standings, and match events. To ensure the quality and reliability of the application, I integrated an MSTest project for comprehensive unit testing, utilizing Moq for dependency mocking and System.Text.Json for seamless JSON serialization. My role encompassed not only the implementation of responsive UI components but also the orchestration of API interactions to present real-time data dynamically. This project honed my skills in full-stack development, API integration, and test-driven development within a modern web application context

Languages: C#, JavaScript, HTML, CSS, JavaScript
Frameworks: Blazor WebAssembly, MSTest, Moq, .NET 8
Programming Concepts and Techniques: Dependency Injection, Asynchronous Programming, LINQ, Timer

Football Data Analysis

In my comprehensive data analysis third-year project, I examined the 2018/2019 English Premier League season to determine the impact of playing at home versus away on team performances. As an avid football enthusiast, I leveraged my passion for the sport to delve into a rich dataset, focusing on key performance metrics such as goals, shots, and corners for both home and away games. Utilizing advanced statistical techniques, I conducted an in-depth exploratory analysis, ensuring data integrity and creating focused subsets for a nuanced examination. My approach involved innovative visualizations and hypothesis testing, including the use of chi-squared tests. My findings revealed a significant 'home advantage', with teams performing notably better in home games across various metrics. This project not only honed my analytical and statistical skills but also demonstrated my ability to derive meaningful insights from complex datasets, offering valuable perspectives in sports analytics.

Languages and Tools: Python, Jupyter Notebook
Libraries: Pandas, Matplotlib, Seaborn
Statistical Techniques: Descriptive Statistics, Chi-Squared Goodness-of-Fit Test
Data Analysis Skills: Data Cleaning, Data Manipulation, Data Visualization
Project Focus: Sports Analytics, Performance Analysis, Home Advantage in Football
